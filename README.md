# [それって"普通"ですか？](https://kmch4n.github.io/Is_that_normal/)
## 導入 "普通"とは何か
- "普通"は固定化された価値ではなく、**社会・他者・自分**の相互作用から生まれる関係的な概念。  
- その"普通"は、テクノロジーによって**常に再設計・破壊・再構築**されてきた。  
- 産業革命が「肉体」を代替したように、AI革命では「思考」が代替される。  
→ 未来の自分の生活のために働く。  
→ 「考えること」がAIに代替される時代、**人間にとって"普通"とは何か？**  

  

## "普通"の構成要素
"普通"を構成する「社会・他者・自分」の相互作用は、具体的に3つの領域(法・市場・世間)で発生している。  
1. 「法」が定義する"普通"（社会）: 法律や制度が定める最低限のライン（例：義務教育、納税、法定労働時間）。これは国家権力による「公的な"普通"」である。  

2. 「市場」が定義する"普通"（他者との比較）: 市場経済における「平均」や「中央値」（例：平均年収、平均的なキャリアパス）。これは経済合理性に基づく「相対的な"普通"」であり、人々が「人並み」を意識する際の最大の参照先である。  

3. 「世間」が定義する"普通"（自分と他者の関係）: 所属するコミュニティの同調圧力や「暗黙の了解」（例：「良い年齢なのだから結婚すべきだ」「残業するのが当たり前だ」）。これは情緒的な「ローカルな"普通"」である。  

  

## 現在の"普通"とその背景
- 現代社会では、「働くことを通じて生活を成り立たせ、安定と安心を得る」という生き方が広く共有されており、それが"普通"であった。  
- 経済的な成功は、長らく「努力」や「幸福」の象徴とされてきた。  
- 日本社会を含む多くの国では、「労働」は単なる経済活動ではなく、「人としての正しさ」や「社会貢献」と直結した道徳的な行為とみなされる。  
- このような価値観は、産業社会以降の「労働中心の仕組み」や「生産性の向上を前提とした社会設計」に支えられてきた。  
- しかし、AIの進化はその前提<u>「働くことが生きることの中心である」</u>という考えを問い直しつつある。  

  

## 未来での労働はどのように変化するか
- 2030年代：**労働時間の平均が段階的に短縮**。(週30h→20h)  
→ ワークライフバランスが大幅に改善される。 週休4日  
→ 今まで以上に、余暇を過ごす時間が長くなる。  

> [!NOTE]
> 2030年代には、人々は週休4日が当たり前になり、空いた時間で「もう一つの人生」を生き始める。  
> 平日は会社員として働き、週末はミュージシャンや農家、ボランティアとして活動する人が増えていく。  
> 仕事は単なる“収入の手段”ではなく、“自分を表現する場”へと変わっていく。  
> AIが事務作業や分析を肩代わりすることで、人間はより創造的で、人と関わる時間に集中できるようになる。  
> 誰もが「本当にやりたかったこと」に一歩踏み出せる時代——  
> それが2030年代の“働く”ということの新しい意味になる。  

<u>業務は大幅に効率化されるものの、まだ人間の意思決定に大幅依存 </u>  

- 2040年代：基礎所得＋公共配当が常設化、**働かなくても衣食住は担保**が標準に。  
働くのは選好へ。(ベーシックインカムの実現)  
→ 極論、"贅沢"を必要としなければ労働の必要性が無くなる。  

> [!NOTE]
> 2040年代には、生活が保障される社会が実現し、人々は“生きるため”ではなく“表現するため”に働くようになる。  
> 絵を描くこと、地域の森を守ること、子どもに物語を語ること——  
> そんな小さな行為が、社会全体の豊かさを形づくる“新しい労働”として価値を持つ。  
> 人々は自分の好奇心をそのまま社会貢献に変え、AIはその挑戦を支える相棒となる。  
> 「好きなこと」と「誰かの役に立つこと」が重なり合う世界。  
> そこでは、働くことが義務ではなく、喜びの循環となっていく。  

<u>働かなくても生きていける時代は、モチベーションや目的意識を失うリスクがある。</u>  

> [!WARNING]
> これらの2030年、2040年論はかなり楽観的な推測である。  
> ベーシックインカムの実現には大幅な生産性の上昇・財源と再分配合意・政治経済的摩擦の解消が不可欠である。  
> → 企業のAI使用に対する適切な課税、技術の進化による人間の価値観の変化によって、実現する可能性もある。  

  

## ベーシックインカムは何をもたらすか
- 「余暇の創造的爆発」の可能性  
→ 古代ギリシャの市民が哲学を発展させたように、余暇が知的・文化的繁栄をもたらす。  
→ ゲーム、スポーツ、アートが「生産性」から解放され、純粋な喜びとして再発見される。  
- 失敗する自由  
→ 起業・創作・研究に挑戦しても生活が保証される。  
→ 「失敗が許される社会」は、イノベーションを加速させる。  
- ケアの充実  
→ 家族の介護、子育て、地域活動に専念できる。  
→ 人間関係の質が劇的に向上する可能性がある。  

  

## 未来での価値の再発見
- AIが生成した音楽は完璧かもしれないが、「目の前で生身の人間が汗を流して演奏するライブ」の熱量には敵わない。価値は「モノ(データ)」から「コト(体験)」へ移行する。  
- "手作り"と言われると付加価値を感じるように、人間が作ったこと、人間が描いたことなどに多くの価値が生まれる。これは、「物」ではなくそのものが持つ「物語」に価値があるとも言える。  
- AIが模倣を極めていく中で、「人間らしい不完全さ」「感情のゆらぎ」が価値を持つ。  
- 人間の生来の姿である、第一次産業(農業・林業・漁業)やアートの分野が盛りあがることが期待できる。  
- 近代以降続いていた"都市集中"がなくなり、田舎暮らしが価値を持つ。  

  

## 「代替困難性」から見る、AIに奪われない仕事
AIやロボットが発展しても、「人間にしかできない仕事」は依然として存在する。  
それは **"予測不能性"、"関係性"、"意味創造性"** の3軸で説明できる。  
| 分類軸 | 内容 | 代表例 |
|----------|--------|-----------|
| 1. **予測不能性**（Uncertainty） | 定型化できない状況への対応・即興的判断 | 救急医療、災害対応、教育、カウンセリング |
| 2. **関係性**（Relationality） | 人と人の間に生まれる信頼・感情の共鳴 | 接客、看護、営業、介護、アート、教育 |
| 3. **意味創造性**（Meaning Creation） | 価値や美を“創り出す”行為 | 研究、芸術、デザイン、ストーリーテリング |

AIは「効率性」では圧倒的に勝るものの、  
「意味を作る」「共感する」「曖昧な状況に対応する」という次元ではまだ人間が優位にある。

  

## 現在と未来の"普通"
> "普通"とは、社会・他者・自分の3つの相互作用から生まれる相対的概念である。  
- テクノロジーによって大きく変わった未来では、これらの3つの状況も大きく変わる。  
→ 現在では"普通"なことも、未来では"普通"では無い。  

| 内容 | 現在(2025年) | 未来(2040年) |
|-------------|-------------|---------------------|
| **なぜ働くか** | 明日の衣食住のため | 自己表現・貢献のため |
| **なぜ学ぶか** | 就職・安定のため | 探究・好奇心・社会課題解決 |
| **なにが幸せか** | 経済的成功・安定 | 自己実現・余暇の充実 |
| **なにを所有するか** | 物・不動産・お金 | 体験・関係・信用・データ |
| **なにが価値か** | 効率・競争・生産性 | 意味・共感・体験の質 |
| **なにがステータスか** | 役職・年収・所属企業 | フォロワー数・信頼度・創造性・コミュニティへの貢献度 |
| **なにが"リスク"か** | 失業・貧困・失敗 | 孤独・無意味感 |
| **"教育"とは何か** | 知識のインプット・正解の暗記 | 問いを立てる力・AIの使いこなし・他者との対話 |


## 本質
- "普通"とは、社会・他者・自分の3つの相互作用から生まれる相対的概念である。  
- テクノロジーが進化するたびに、“普通”は再設計されてきた。  
- また"普通"の概念を破壊するのも、テクノロジーである。  
- 産業革命が「機械による肉体の代替」を生んだように、AI革命は「知能による思考の代替」を起こす。
