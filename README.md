# それって"普通"ですか？
## 現在の人間生活の"普通"とは何か
→ 未来の自分の生活のために働く  


## 実現可能性シナリオ
2030年代：労働時間の平均が段階的に短縮（週30h→20h）。
→ ワークライフバランスが大幅に改善される 週休4日  

2040年代：基礎所得＋公共配当が常設化、“働かなくても衣食住は担保”が標準に。働くのは選好へ。(ベーシックインカムの実現)
→ 極論、"贅沢"を必要としなければ労働の必要性が無くなる
→ 絶対的貧困は無くなったところで、相対的(他者比較)貧困は残り続ける。労働の目的が、"生活"から"付加価値を購入する財"と価値観の変化？

## ベーシックインカムは何をもたらすか
大学進学という選択
→ 大学に行く、という選択をすることが不可能な層に、機会を提供できる
→ その人間が、自分が実際に学びたいことについて学べる
→ 逆に、大学に行く必要性も減るのではないか。就職活動のために大学へ進学する必要はない。

## "普通"と幸福
現在 "普通"の軸は、労働・安定 幸せの形は経済的成功
未来 "普通"の軸は、"何を生み出すか"より"どう生きるか"

## 本質
- "普通"とは、社会・他者・自分の3つの相互作用から生まれる相対的概念。
- テクノロジーが進化するたびに、“普通”は再設計されてきた。
- また"普通"の概念を破壊するのも、テクノロジーである
- 産業革命が「機械による肉体の代替」を生んだように、AI革命は「知能による思考の代替」を起こす。
- "普通"と"正しさ" それが"選択"へシフト
