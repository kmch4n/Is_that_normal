"普通"とは何か

定義（いま）

多くの人にとって“普通”とは、未来の自分の生活のために働くこと。雇用・安定・経済的成功を軸に、人生が設計されています。

  • 学ぶ → 働く → 消費する の直線モデル
  • 幸福＝経済的達成（年収・肩書など）
  • 基準は社会・他者との相対比較に依存

未来での労働はどのように変化するか

2030年代：労働時間が段階的に短縮

平均労働時間が 週30時間→20時間 へ。週休4日 が普及し、ワークライフバランスが大幅に改善。

2040年代：衣食住が担保される“普通”

基礎所得＋公共配当が常設化し、働かなくても衣食住は標準で担保。働くことは選好へ。

  • “贅沢”を求めなければ、労働は不要に近づく
  • 絶対的貧困は縮小するが、相対的貧困は残る課題
  • 労働の目的が「生活維持」→「付加価値の購入」にシフト

ベーシックインカムは何をもたらすか

ポジティブな変化

  • 大学進学の機会が拡大し、「本当に学びたいこと」にアクセス
  • 就職のためだけの進学からの解放 → 多様な学びの軸
  • 挑戦コストの低下：起業・創作・地域活動がしやすくなる

生まれる問い

  • 「大学に行く必要はある？」資格の意味づけが再編
  • 相対的な格差認知と承認欲求の再設計
  • 勤労インセンティブや税制・財源設計の新たなバランス

AI時代に生まれる“価値”

人間の“ゆらぎ”が価値

AIが模倣を極めるほど、不完全さや感情のゆらぎにプレミアムがつく。

手作り / 生演奏 / 直筆

体験の自給

農家でなくても「自分の食べ物を自分で作る」といった体験価値が拡張。

共同菜園 / メイカースペース

編集の時代

誰もがAIと共創し、何を生み出すか → どう生きるかへ軸が移る。

選択設計 / 意味づけ

「代替困難性」から見る、AIに奪われない仕事

AI/ロボットが発展しても、次の3軸は人間が中心であり続けます。

① 予測不能性

定型化できない状況への即興的判断。

  • 救急医療・災害対応
  • 教育・カウンセリング

② 関係性

人と人の信頼・感情の共鳴を扱う。

  • 看護・介護・接客・営業
  • アート・教育

③ 意味創造性

価値や美を"創り出す"営み。

  • 研究・デザイン・物語制作
  • 芸術・キュレーション

現在と未来の"普通"と幸福

現在の軸

  • 労働・安定が中心
  • 幸せ＝経済的成功という物差し

未来の軸

  • 「何を生み出すか」より「どう生きるか」へ
  • 時間と注意の再配分が幸福度を決める

ベーシックインカム時代の“お金”と報酬

衣食住は“標準装備”へ

  • 基礎所得＋公共配当で生存のための賃金の必然性が低下
  • 働くことは「義務」から「選択」へ

新しいインセンティブ設計

  • 不人気だが社会的に必要な仕事には高報酬＋リスペクト
  • お金＝自己実現・貢献の指標（多層通貨の可能性）

Key Message

“普通”とは、社会に従うことではなく、自分の「存在」を選び取ることだ。テクノロジーがすべてを自動化する時代、生きること自体が人間の創造行為になる。

本質：”普通”は相互作用から生まれる

  • “普通”は 社会・他者・自分 の相互作用による相対概念。
  • テクノロジーの進化のたびに“普通”は再設計されてきた。
  • 産業革命が「肉体の代替」を生んだように、AI革命は「思考の代替」を加速。
  • “普通”と“正しさ”は固定値ではなく、選択の設計へと移行する。
